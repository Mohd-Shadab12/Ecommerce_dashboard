📊 E-Commerce Sales Dashboard (Power BI)

📌 Overview

This project presents a 3-page interactive Power BI dashboard analyzing global e-commerce sales data.

🚀 Key Features

- Sales Performance Overview (Revenue, Profit, Orders, Trends)
- Customer & Profit Analysis (Discount impact, Segmentation)
- Regional & Payment Insights (Country performance, Shipping cost, Payment methods)

📊 Tools Used

- Power BI
- Excel
- Data Visualization

📈 Key Insights

- Higher discounts reduce profitability
- Consumer segment generates highest profit
- Europe & North America are most profitable regions

📂 Files Included

- Power BI Dashboard (.pbix)
- Dataset (.csv)
- Dashboard Screenshots
